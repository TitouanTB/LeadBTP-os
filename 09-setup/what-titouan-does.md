# Ce que Titouan doit faire

## 1. Fixer l'environnement technique
- Choisir stack app/web
- Créer repo GitHub principal
- Préparer dev/staging/prod
- Préparer hébergement (ex: Vercel / autre)
- Préparer DB / auth / stockage

## 2. Préparer les accès
- GitHub
- hébergement
- DB
- SMS / téléphonie
- email
- analytics
- éventuellement scraping/API

## 3. Créer les secrets avec séparation des rôles
- éviter de donner toutes les clés à tous les agents
- préférer des accès par périmètre

## 4. Configurer Git local
- git user.name
- git user.email

## 5. Valider les règles d'autonomie
- ce que les agents peuvent faire seuls
- ce qui demande validation
