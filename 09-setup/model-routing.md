# Model Routing

## Premium model
Utiliser OpenAI/Codex / GPT-5.x pour:
- Napoleon
- architecture CTO
- specs produit clés
- copy stratégique
- code complexe
- arbitrages importants

## Cheap/Fast model
Utiliser Gemini Flash (ou équivalent) pour:
- résumés simples
- transformations de texte
- extraction/formatage
- QA légère
- tâches mécaniques
- brouillons simples

## Rule
Les tâches à faible enjeu et faible ambiguïté doivent basculer sur le modèle cheap.
Les tâches à fort enjeu, créatives ou structurantes restent sur le modèle premium.
