# LeadBTP Studio OS

## Mission
Transformer LeadBTP en studio IA de production capable de construire :
- des features SaaS
- des landing pages
- des sites clients
- des assets marketing

avec un workflow clair : brief → cadrage → production → review → preview → itérations → validation.

## Principe
Titouan donne l'objectif.
Napoleon orchestre.
Les pôles internes produisent.
Tout laisse une trace.
Tout doit pouvoir être revu.

## Pôles
- Napoleon
- Product
- CTO / Dev SaaS
- Website Factory
- Growth
- Ops / QA

## Règles
- une demande = un brief clair
- un brief = une production suivie
- une production = une review
- une review = des itérations traçables
- toute sortie importante est documentée
- toute action sensible passe par validation
