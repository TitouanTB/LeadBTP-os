# LeadBTP - Vision produit détaillée

## Vision
LeadBTP est un SaaS/agence hybride qui vend des systèmes clé-en-main aux artisans indépendants français. Le produit combine :
- un site web professionnel optimisé conversion
- des automatisations SMS/email
- une interface mobile qui centralise la relation client

## Positionnement
- SaaS + Agence : configuration faite pour le client
- anti-jargon tech
- ROI visible et chiffré

## Utilisateurs
### Admin (Titouan)
- configure les clients
- déploie les sites
- surveille le système
- usage quotidien desktop

### Artisan client
- gère ses leads
- répond aux messages
- consulte son ROI
- usage quotidien mobile

## Tunnel d'acquisition
Site offert à vie + 15 jours d'essai Pro gratuit.
Fin d'essai :
- Option A : 297€/mois
- Option B : 197€ mois 1 contre 1 referral qualifié puis 297€
- Option C : mois 1 offert contre 3 referrals qualifiés puis 297€

Un referral qualifié = artisan indépendant qui accepte un appel de 15 min.

## Points clés retenus
- Le système doit travailler sans intervention pendant que l'artisan est sur chantier
- Le client ne doit pas avoir à configurer l'outil
- Le dashboard doit justifier l'abonnement par le ROI
- Le site est le point d'entrée principal du système
- Le client doit percevoir un rendu agence, pas template générique
- Le pipeline site → webhook → lead → inbox → SMS est central
- La PWA mobile est une priorité produit forte
- Le tracking referral LeadBTP et campagne anciens clients est prioritaire juste après le MVP
