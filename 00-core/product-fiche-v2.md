# LeadBTP - Fiche Produit V2

## Positionnement
LeadBTP est un SaaS + agence opérée pour artisans BTP. On vend un système qui capte les leads, évite les appels perdus, relance les devis, centralise les échanges et montre le ROI.

## Promesse
On récupère vos leads, on relance vos devis, et on vous aide à signer plus de chantiers pendant que vous êtes sur le chantier.

## Coeur produit
1. Site Engine intégré au SaaS
2. SaaS Core (leads, inbox, automations, pipeline, ROI)
3. Admin / Studio pour produire, reviewer et déployer

## MVP
- auth
- clients
- intake client premium
- site lead-gen intégré
- form → lead
- inbox simple
- SMS appel manqué
- relances devis
- pipeline leads
- dashboard ROI
- admin minimal
- preview/deploy Vercel

## Priorités fortes
- site = point d'entrée principal
- rendu agence, pas template cheap
- mode création site complet + mode optimisation site existant
- mobile-first côté artisan
- PWA prioritaire dans la trajectoire produit
