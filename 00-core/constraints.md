# Constraints

## Produit
- mobile-first
- simple pour artisans non techniques
- résultat visible rapidement
- éviter l'effet usine à gaz

## Design
- crédible
- moderne
- non générique IA
- orienté conversion

## Opérations
- agents par périmètre
- pas d'accès universel total
- actions sensibles sous validation
- traçabilité par fichiers + git

## Coût
- limiter le contexte inutile
- préférer dossiers structurés et handoffs courts
- lancer les sous-agents à la demande
