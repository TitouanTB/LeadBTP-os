# Access Matrix

## Napoleon
- Read: tout
- Write: docs, plans, mémoire projet
- External actions: validation requise pour actions sensibles

## Product Lead
- Read: core + product + playbooks
- Write: specs, flows, roadmap
- No prod deploy

## CTO Lead
- Read: core + product + tech
- Write: architecture, code, infra docs
- Dev/staging: oui
- Prod: validation pour changements majeurs

## Frontend Lead
- Read: core + product + site-factory
- Write: templates, composants, UI
- Preview deploy: oui
- Prod direct: non par défaut

## Growth Lead
- Read: core + growth + playbooks
- Write: campagnes, lists, scripts, creative
- Real sends / scraping sensible: validation

## Ops/QA Lead
- Read: tout
- Write: SOPs, rapports, checklists
- No prod destructive actions
