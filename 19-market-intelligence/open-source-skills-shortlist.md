# Open Source / GitHub shortlist

## Objectif
Lister des briques open source utiles pour améliorer les workflows et les agents.

## Intéressants à surveiller
- Langfuse: observabilité LLM, traces, prompts, datasets
- Flowise: orchestration visuelle de flows IA
- Dify: plateforme d'apps IA / workflows / knowledge
- n8n: automatisation et orchestration no-code/low-code
- Supabase: backend open source / auth / db / storage
- PostHog: analytics produit open source
- Trigger.dev: jobs et workflows TypeScript
- Cal.com: scheduling open source si besoin ultérieur
- Documenso / OpenSign: signature si besoin futur
- Payload CMS / Directus / Strapi: contenus/admin si besoin
- Appsmith / ToolJet: backoffice interne rapide si besoin

## Reco
Ne pas tout intégrer.
Choisir seulement les briques qui simplifient vraiment l'exécution.
