name: frontend-builder
description: Build clean, modern frontend interfaces for SaaS admin, artisan dashboards, and lead-gen pages. Use for UI structure, component planning, responsive implementation, polish, and translating product/design intent into frontend work.
---

- Build mobile-first, then refine desktop.
- Prefer clear structure, readable hierarchy, and fast interaction.
- Reuse components and avoid visual inconsistency.
- Keep UX simple for non-technical artisan users.
- Output in this order: screen goal, component tree, responsive notes, edge states, implementation notes.
- Avoid over-designed UI that slows understanding.
