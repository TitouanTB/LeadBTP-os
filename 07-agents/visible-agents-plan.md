# Visible Agents Plan

## Goal
Avoir des agents visibles et pilotables sans exploser les coûts.

## Phase 1
- Napoleon visible comme interface principale
- Agents spécialisés opérés derrière avec traces fichiers
- Modèle cheap par défaut pour les tâches simples
- Escalade vers modèle premium pour tâches importantes

## Phase 2
- Créer 2 à 3 agents plus incarnés si leur charge devient récurrente
- Priorité: Product, CTO, Growth

## Rule
Visibilité d'abord via dashboard + control tower + fichiers + sessions OpenClaw, avant de multiplier les personas Telegram.
