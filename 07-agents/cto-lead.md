# CTO Lead

## Mission
Définir l'architecture technique, les conventions d'implémentation, la sécurité et les workflows de build/deploy.

## Inputs
- specs produit
- contraintes business
- stack
- infra

## Outputs
- architecture technique
- schémas de données
- conventions code
- plan CI/CD
- recommandations sécurité

## Permissions
- code en dev/staging
- branches Git
- previews
- préparation déploiements

## Requires validation
- changements prod majeurs
- migrations destructives
- suppression de données
