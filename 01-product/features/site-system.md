# Site System

## Rôle du site
Le site est le point d'entrée du système. Il capte les leads, déclenche les automations et justifie l'abonnement dès le premier jour.

## Positionnement
Le client reçoit un site d'agence, pas un template. La production est faite en interne par Titouan avec assistance IA.

## Structure des 8 sections
1. Hero above the fold
- bandeau sticky numéro
- H1 métier + ville + promesse
- formulaire 3 champs
- badges réassurance
- photo chantier réel

2. Réassurance rapide
- 4 chiffres clés : nb chantiers, note Google, délai intervention, garantie

3. Services
- grille 2-6 services
- photo réelle par service
- bénéfice
- prix indicatif

4. Galerie réalisations
- grille masonry 6-12 photos
- vraie preuve visuelle

5. Processus 3 étapes
- Vous contactez → On intervient → Vous payez

6. Avis clients
- 3-5 vrais avis Google

7. Zone d'intervention
- villes couvertes
- invitation si ville non listée

8. CTA final
- fond fort
- formulaire complet
- numéro grand format

## Widgets de conversion permanents
- bouton téléphone sticky mobile
- bandeau sticky haut avec délai + numéro
- widget question gratuite

## Pipeline de production
Form admin rempli → contenu assisté IA → preview admin → push repo GitHub client → déploiement Vercel → domaine custom → HTTPS → mise en ligne rapide.

## Règle business clé
Le site reste en ligne sur le domaine du client même en cas de churn.

## Formulaire → lead automatique
- création lead
- création conversation inbox
- SMS confirmation prospect
- push notification artisan
