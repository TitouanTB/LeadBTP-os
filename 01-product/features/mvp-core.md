# MVP Core Features

- Site lead-gen pour 3 métiers prioritaires
- Formulaire de qualification lead
- Inbox simple centralisée
- SMS automatique après appel manqué
- Relances devis simples
- Reporting ROI minimal

## Non-MVP
- multi-users avancé
- intégrations complexes
- moteur de génération totalement libre
- growth automation trop large
