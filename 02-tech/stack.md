## Recommandation initiale
- Frontend/App: Next.js
- Backend/API: Next.js server routes ou backend Node dédié selon complexité
- DB: Postgres
- Auth: solution simple et standard
- Storage: objet simple pour assets
- Deploy: Vercel ou équivalent pour vitesse au départ
- Jobs/queues: à ajouter seulement si nécessaire

## Principe
Démarrer simple, éviter la sur-ingénierie, séparer les environnements.
